Project Resource Attributions

In compliance with academic integrity standards and course development requirements, the following credentials document the data sources, textual quotes, and visual assets utilized across the Global Trade Center (GTC) portal.

1. Literary and Typography References

  • Inspirational Epigraph: The textual excerpt utilized in the project guidelines and featured during introductory prompts is from the address "Happiness, Your Heritage" by Dieter F. Uchtdorf (October 2008).
  • Typography Layout: Font pairings and fallbacks ("Helvetica Neue", Helvetica, Arial, sans-serif) are styled in accordance with standard cross-platform web safety typography guides.

2. International Trade Frameworks & Regulatory Data

  • EU Access2Markets Data Structure: Regulatory criteria, Harmonized Tariff Schedule (HTS) data scopes, and Rules of Origin (RoO) logic models are adapted from the official European Commission's Access2Markets Portal.
  • International Trade Centre (ITC) Concepts: Diagnostic methodology tracks and operational parameters used inside the Local Storage readiness quiz are structurally modeled around information provided by the International Trade Centre (ITC) Academy.
  • Global Customs Clearing Schemas: Regional parameters (such as the Angolan Documento Único via ASYCUDA networks and Canadian CARM systems) reflect real-world automated logistics interfaces.

3. Visual Assets and Vector Icons

  • Hero Image Asset / Open Graph Banner: The corporate trade graphic backdrop (gtc-og-banner.webp) is optimized for web delivery using creative commons assets compiled from Unsplash.
  • Interface Hamburger Icon & Vectors: Responsive layout navigation bars and modal close nodes are custom vector-constructed natively via semantic CSS styling rules to guarantee 100% scores on Lighthouse accessibility audits.
  • Favicon Branding Asset: The global branding mark icon (favicon.ico) was compiled using web-safe graphic asset generation tools.

4. Peer and Technical Consultations

As restricted by the individual exam guidelines, this project was developed entirely independently. No third-party frameworks (such as TailwindCSS or Bootstrap) or external drag-and-drop tools were used. The code was validated locally using Google Chrome Developer Tools (Lighthouse Diagnostics, CSS Overview, and Console Audits).
